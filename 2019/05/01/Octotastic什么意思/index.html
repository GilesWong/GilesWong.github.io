Having an … Octastic day?

今天在提交申请 Github Student Pack 之后，出现了一个这个界面：

正常情况下这个界面应该被随手关掉才对，但是习惯性的读了一遍之后，发现结尾这个 Oct…tatos…不对…Octotastic 拼写起来就有点困难的单词似乎没见过。 顺手选中，三指点按…

似乎是个没见过的词呢

搜索了一下，大概呢意思就是: -tastic词缀听起来是一种情感的流露，是一种非正式的写法。

那么流露的是什么感情呢？

-tastic前面是Octo…octopus？？？ 想了下GitHub的吉祥物，对没错就这厮：

Github 的吉祥物名字叫 Octocat(章鱼🐱？)，也就是说这个Octotastic是个GitHub自己造的词。

那 Have an octotastic day 的意思是 有个很Github的一天？？(摸着我的光头想了想)

补：搜索到了一个因缺思厅的翻译

所以 Octotastic 代表的是 Github 官方情感的流露（也可能也是一种对海洋生物的关注的流露），于是 Have an Octotastic day! 可以理解为：~~祝愿您今天在全球最大同性交友网站找到真爱！~~ 祝您今天交友愉快！
